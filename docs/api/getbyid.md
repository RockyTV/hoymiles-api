### Get station info

This endpoint retrieves general information about the current solar station, as well as information about its owner and the person responsible for its installation.

- URL: `/zhgf-core/api/0/powerStation/getById`
- Method: **POST**
- Headers: 
  - Authorization
- Params:
  - **id**: string
